# DS Salary Predictor — Web Interface

A clean, dark-themed Flask web app that wraps the ML salary prediction model
trained in `project_ml.ipynb`.

## Quick Start

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Train the model (if not done yet)
Open `project_ml.ipynb` in Jupyter or Google Colab and run all cells.
This generates four artefact files:

| File | Contents |
|---|---|
| `salary_model.pkl` | Trained Random Forest (200 trees) |
| `scaler.pkl` | StandardScaler (used for Linear Regression path) |
| `label_encoders.pkl` | LabelEncoders for categorical columns |
| `feature_names.pkl` | Ordered list of feature column names |

Place all four files in the **same directory** as `app.py`.

### 3. Run the web server
```bash
python app.py
```
Then open your browser at **http://127.0.0.1:5000**

## API Reference

### `POST /predict`
Accepts JSON, returns salary breakdown.

**Request body**
```json
{
  "work_year": "2023",
  "experience_level": "SE",
  "employment_type": "FT",
  "job_title": "Data Scientist",
  "remote_ratio": "100",
  "company_size": "M"
}
```

**Experience level codes**
| Code | Meaning |
|---|---|
| EN | Entry-level |
| MI | Mid-level |
| SE | Senior |
| EX | Executive / Director |

**Employment type codes**
| Code | Meaning |
|---|---|
| FT | Full-time |
| PT | Part-time |
| CT | Contract |
| FL | Freelance |

**Company size codes**
| Code | Meaning |
|---|---|
| S | Small (< 50 employees) |
| M | Medium (50–250 employees) |
| L | Large (> 250 employees) |

**Success response**
```json
{
  "annual": 145000,
  "monthly": 12083,
  "weekly": 2788
}
```

**Error response**
```json
{ "error": "Human-readable error message" }
```

### `GET /health`
Returns `{ "model_loaded": true/false }`.
