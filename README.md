# 🤖 IEEE AI/ML/DL Internship — Machine Learning Projects

This repository contains four end-to-end Machine Learning projects developed during the **IEEE AI/ML/DL Internship Program**.

---

## 📂 Projects

### 1. 🛍️ Customer Segmentation
> Unsupervised Learning · K-Means Clustering · Streamlit

Groups mall customers by Annual Income and Spending Score using K-Means and Hierarchical Clustering. Provides interactive cluster visualizations and business insights.

📁 [View Project →](./Customer_Segmentation/)

**Tech:** Python · Scikit-Learn · Streamlit · Pandas · Seaborn

---

### 2. 🩺 Diabetes Risk Prediction
> Supervised Learning · Logistic Regression · Classification

Predicts whether a patient is at risk of diabetes from 8 clinical health parameters. Includes probability score, risk meter, and health recommendations.

📁 [View Project →](./Diabetes_Prediction/)
🌐 [Live App →](https://diabetespredictiongit-inql8uanwywdaxkf5wwjnb.streamlit.app)

**Tech:** Python · Scikit-Learn · Streamlit · Pandas

---

### 3. 🌿 Plant Disease Detection
> Deep Learning · CNN · Image Classification

Detects plant diseases from leaf images using a trained Convolutional Neural Network. Provides disease diagnosis and treatment suggestions.

📁 [View Project →](./Plant_Disease_Detection/)

**Tech:** Python · TensorFlow · Keras · Streamlit · Pillow

---

### 4. 💰 Data Science Salary Predictor
> Machine Learning · Random Forest Regression · Flask

Predicts Data Science salaries based on job role, experience level, employment type, company size, and remote work ratio.

📁 [View Project →](./salary_predictor/salary_predictor/)

## ⚙️ Quick Start

```bash
git clone https://github.com/Ak-oo7/IEEE-ML-Projects.git
cd IEEE-ML-Projects

# Run any project
cd Customer_Segmentation && pip install -r requirements.txt && streamlit run app.py
# or
cd Diabetes_Prediction && pip install -r requirements.txt && streamlit run app.py
# or (use conda — see Plant_Disease_Detection/README.md)
cd Plant_Disease_Detection && streamlit run app.py
```
